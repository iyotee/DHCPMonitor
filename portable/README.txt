DHCP Monitor Pro - Portable Version 
================================== 
 
This is a portable version of DHCP Monitor Pro. 
 
Requirements: 
- Windows 10 or later 
- Npcap installed (https://npcap.com/) 
- Administrator privileges to capture packets 
 
Usage: 
1. Right-click "DHCP Monitor Pro.exe" 
2. Select "Run as administrator" 
3. Select your network interface 
4. Start capturing DHCP packets 
 
Note: This portable version includes the required Npcap DLLs.
